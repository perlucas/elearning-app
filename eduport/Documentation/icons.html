Font icons

Retina-ready font icons with a very modern touch that will beautifully compliment your project.

We have included Font awesome and Bootstrap Icons fonts in Eduport theme.

Font awesome:

You can import and use any Font Awesome icons just by importing it from react-icons/fa and using the imported Icon anywhere in your return statement, and the icon will render as an SVG.

  1. Its name, prefixed with Fa (meaning “font awesome” naturally!)

import { FaFacebookF, FaGoogle } from "react-icons/fa"

<FaFacebookF /> // Facebook Icon
<FaGoogle  /> // Google Icon

A complete list of Font awesome icons along with its names can be found here

Bootstrap Icons:

You can place Bootstrap Icons just about anywhere using icon's name. See below example code:

  1. Its name, prefixed with BSIcon (meaning “Bootstrap Icons” naturally!) 

  import { BsArrowUp, BsPlus } from "react-icons/bs"

  <BsArrowUp />   // Arrow Up Icon
  <BsPlus />   // Plus Icon

A complete list of Bootstrap icons along with the class names can be found here
