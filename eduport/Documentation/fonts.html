Fonts

Learn how to change site fonts

By default, theme uses Google fonts. In case, If you want to change font to any other Google fonts, Please have a look in the head part of the index.html file and you will find these tags: 

<!-- Google Font -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Heebo:wght@400;500;700&family=Roboto:wght@400;500;700&display=swap" rel="stylesheet">

To change the font, first go to Google fonts, choose your desired font and use the generated code.

After changing fonts you need to update font families in _variables.scss or override in _user-variables.scss file 

$font-family-base:           'Roboto', sans-serif;
$headings-font-family:        'Heebo', sans-serif;

If you want to use custom fonts, remove tags completely and visit Self Hosted Fonts
